# Go Learning Repository

A comprehensive collection of Go programming examples organized by complexity level, covering everything from fundamentals to few advanced concepts.

## 📁 Repository Structure

### Fundamentals
- **helloworld.go** - Basic introduction to Go with data types (integers, unsigned integers, strings, etc.), type conversions, and math operations

### Basics
- **functions.go** - Function fundamentals including multiple return values, anonymous functions, and functions as parameters
- **datastructures.go** - Arrays, slices, and maps with various operations
- **errorHandling.go** - Error handling patterns and best practices
- **structsAndInterfaces.go** - Custom types (structs), methods, embedded structs, and interface implementations

### Advanced
- **goroutines1.go** - Introduction to goroutines and channels for concurrent programming
- **goroutines2.go** - Advanced goroutine patterns using unidirectional and buffered channels
- **goroutines3.go** - Further goroutine examples covering mutex and wait group
- **pointersAndReferences.go** - Working with pointers, pass-by-value vs pass-by-reference, and pointer methods
- **generics.go** - Generic functions and types for writing reusable code across different data types

## 🚀 Getting Started

### Prerequisites
- Go 1.26.2 (or compatible version)

### Running Examples

#### From the Fundamentals folder:
```bash
cd fundamentals
go run .
```

#### From the Basics folder:
```bash
cd basics
go run .
```

#### From the Advanced folder:
```bash
cd advanced
go run .
```

## 📝 Notes

- Each example file contains detailed comments explaining the concepts
- Run examples one at a time, especially goroutines examples, to clearly observe the behavior
- Modify and experiment with the code to deepen your understanding
